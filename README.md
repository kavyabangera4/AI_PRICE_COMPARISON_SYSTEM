## ğŸ”„ System Workflow
1. User logs in or signs up
2. User searches product using text or image
3. AI models generate embeddings
4. Similarity matching retrieves relevant products
5. User can:
   - Add to wishlist
   - Set price alerts
   - View price history
6. Admin replies to user queries
7. Email notification is sent when price alert is triggered
